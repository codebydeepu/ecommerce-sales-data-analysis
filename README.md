# E-Commerce Sales Data Analysis

This project analyzes an e-commerce sales dataset using Python

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
- Data cleaning
- Sales by region
- Top selling products
- Monthly sales trends
- Profit analysis

## Key Insights
- West region generates the highest sales
- Technology category produces the most profit
- Canon imageCLASS 2200 Advanced Copier is the top selling product

## Visualizations
Bar Chart – Sales by Region  
Line Chart – Monthly Sales Trend  
Pie Chart – Sales Distribution by Category
